Como Decidi Virar Dev?

Durante o último módulo do curso de Mecatrônica no SENAI, meu grupo teve o TCC realocado, saindo de "Melhoria de Processos" para "Internet das Coisas (IoT)", nesse caso, começamos a trabalhar com Python, OpenCV, Pandas e CFTV, mudando totalmente o rumo das coisas.

Desde então, de forma autodidata, venho estudando programação com objetivo de ampliar o conhecimento (tanto de processos de automação) como para ingressar no mercado de TI e Software Development.

Atualmente, estou tentando aprender as hardskills necessárias para ser um Dev FullStack, utilizando Python, C, JavaScript, HTML e CSS. Também tentando absorver noções de Bancos de Dados (SQL e NoSQL) e Frameworks (como Django, Flask e Flet) para desenvolvimento de aplicações/web.
